# ThreatLens AI

## Overview

ThreatLens AI is an AI-Powered Threat Intelligence and OWASP Risk Analysis Platform designed to analyze IP addresses, URLs, domains, and file hashes using multiple threat intelligence sources.

## Features

- IOC Enrichment
- VirusTotal Integration
- AbuseIPDB Integration
- AlienVault OTX Integration
- Threat Scoring
- AI-Powered Threat Summaries
- OWASP Top 10 Mapping
- MITRE ATT&CK Mapping
- CVE Correlation
- Threat Trend Visualization

## Tech Stack

### Backend
- FastAPI
- Python

### Frontend
- Next.js

### Database
- SQLite

## Goal

To provide actionable threat intelligence and risk analysis for cybersecurity investigations.
